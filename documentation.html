If-elif-else Conditions

If, elif and else conditions are the most basic parts of of coding, not just in Python but in all programming languages althpugh their names differ in other languages. (eg. elif is else if in Java)These conditions direct the code and tells it what to do. If option A can't be done do option B. If option B can't be done either, got to option C.

The way it works is that, you start the conditioning with if clause. You can use multiple elif conditions for any number of possibilities yo need. But you can only use one if and one else for one sitiuation. You don't need to use else at all, Python lets you use elif if you want. Here's a simple example:

number = 1

If number == 0:
  print("The number is 0.")
elif number == 1:
  print("The number is 1.")
else:
  print("The number is not 1 or 0.")

As you can clearly see the output will be: The number is 1.



For and While Loops

For and while loops re when you want to keep doing something. With for loops, you can limit the number of times the loop runs. However while loops are used with a condition. The loop works while the condition is true and if there isn't another condtions in the loop to change the first condition, the loop doesn't stop nad runs until the computer is forced to stop. Here are two simple examples:

for i in range(5):
  print(i)

In for loops if the start of the loop is not specified, it will start with zero and print as many numbers as the range. So naturally because it started with 0, 5 will not be printed. Aside from start and end, for loops can have a skipper. For example;

for i in range(1,2,6):
  print(i)

will print:
    1
    3
    5

Functions

Functions are one of the most important components of not only Python but all programming languages. They are a way of not repeating the same code but using it with different values. Here's a very simple function example:

def function1(number):
  print("The number is "+number+".")

In this example, number is a variable. If the value of the variable is 5, the output will be "The number is 5.". However if it's 10, the output will be "The number is 10."

Pandas Library

Pandas is a Python library, used for manuplating and analyzing data. With Pandas library you can:

  • Read CSV, Excel and JSON files.
  • Clean data.
  • Merge data.
  • Create Pandas series and dataframes.
  • Correlate data.

Here's a simple Pandas code:

import pandas as pd
df = pd.read_csv('data.csv')
print(df.to_string())
Matplotlib Library

Matplotlib is a Pyhton library used for making plots which are commonly used in Data Science in order to show the data in a more understandable way to the target audience. With this library you can plot histograms, line-plots etc. Here's how you can code a simple line diagram:

import matplotlib.pyplot as plt
import numpy as np
xpoints = np.array([0, 6])
ypoints = np.array([0, 250])
plt.plot(xpoints, ypoints)
plt.show()
